# NeuroLab Success Criteria

## MVP success (what “good” means)

### Latency

- **&lt;5–15 seconds** for typical meta-analytic map retrieval + rendering preview.

### Scientific hygiene

- **100% of maps** include:
  - Evidence Tier label (A / B / C)
  - Data source + algorithm + version
  - Spatial null model method for map–map correlations (where relevant)

### Adoption

- Users can answer basic map questions (term→map, drug comparison, receptor overlay) **without** knowing fMRI preprocessing or coordinate systems.

### Trust

- Clear separation of *what is known* vs *what is hypothesized* in every response.
- Tier B/C outputs include “How to validate” (minimum viable experiment, analysis plan, alternative explanations, links to methods when available).

### Integration

- NeuroLab can run standalone or integrate with an external analysis layer; CBMA and map-comparison workflows can execute reproducibly, and the viewer consumes a manifest with Evidence Tier UI.
